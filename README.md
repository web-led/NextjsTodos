# مشروع Next.js لتطبيق المهام اليومية

## المقدمة

يهدف هذا المشروع إلى إنشاء تطبيق ويب لإدارة المهام اليومية باستخدام إطار العمل Next.js. يتميز هذا الإطار بالسرعة والأداء العالي وسهولة التطوير والتوسع.

## الميزات

- إضافة المهام الجديدة وعرضها في قائمة المهام.
- تحديد حالة المهمة (مكتملة / غير مكتملة) وتحديثها.
- حذف المهام التي تم إنجازها.
- تصفية المهام حسب حالتها (مكتملة / غير مكتملة).
- تخزين المهام في قاعدة بيانات MongoDB.

## التثبيت والاستخدام

1. قم بتثبيت Node.js و MongoDB على جهاز الكمبيوتر الخاص بك.
2. قم بتنزيل المشروع من GitHub.
3. استخدم هذا الأمر لتثبيت جميع الحزم المستخدمة في المشروع:
```
npm install
```
4. قم بتشغيل خادم MongoDB على جهاز الكمبيوتر الخاص بك.
5. استخدم هذا الأمر لتشغيل التطبيق:
```
npm run dev
```
6. قم بفتح المتصفح والانتقال إلى http://localhost:3000 لعرض التطبيق.

## الخاتمة

يمكن تحسين هذا المشروع بإضافة المزيد من الميزات مثل التنبيهات والتذكير بالمهام المهمة وغيرها. ويعد Next.js إطار عمل رائع لتطوير تطبيقات الويب بسرعة وسهولة.
